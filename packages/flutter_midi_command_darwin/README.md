# flutter_midi_command_darwin

[![pub package](https://img.shields.io/pub/v/flutter_midi_command_darwin.svg)](https://pub.dev/packages/flutter_midi_command_darwin)

Darwin (iOS/macOS) host implementation for `flutter_midi_command`.

This package is an endorsed federated implementation and is not intended to be used directly.
Add `flutter_midi_command` to your app; Flutter will select this implementation on iOS/macOS.
